Stabilize your Capital (EtherCash)

Smart Contract

EtherCash is a SmartContract at EtherScale Blockchain made to be a DAILY CURRENCY which pairs with any Fiat Money using ISO 4217 currency code.

Pairing Stability

Each EtherCash abstraction is a stable token pairing to a State Currency, like: 1EtherUSD = 1USD (US Dolar), 1EtherEUR = 1EUR (Euro), and so on...
